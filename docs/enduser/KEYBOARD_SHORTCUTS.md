# Keyboard Shortcuts

## Global

- `Ctrl+O` — Open project
- `Ctrl+S` — Save project
- `Alt+Q` — Switch to Demo mode
- `Alt+W` — Switch to Scene mode
- `Alt+E` — Switch to Post FX mode
- `Alt+SPACE` - Toggle playback

## Preview / UI

- `Alt+F` — Toggle preview fullscreen
- `Alt+D` — Toggle performance overlay

## Debug Aids

- `Ctrl+Shift+K` — Toggle screen-keys overlay (Scene mode)

## Shader Editor

- `Alt+Shift+F` — Auto-indent / format shader code
- `Ctrl+Enter` or `F7` — Compile current shader
- `Ctrl+F` — Toggle Search/Replace bar
